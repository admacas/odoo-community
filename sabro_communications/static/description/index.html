Product Description

A simple, easy to use tool to enable communications to your business. SaBRO Communications integrate Odoo contacts and users to SaBRO platform in order to enable calling features on odoo platform. SaBRO Communications provides all necessary standard and advanced features to enable telephony to your business.

Features

Standard

  1. Incoming Calls
  2. Outgoing Calls
  3. Call Forwarding/Transfer
  4. Music on Hold
  5. Voicemails
  6. Call History

Advanced

  1. Business Numbers
  2. Number Mapping
  3. Caller ID Management
  4. Call Routing
  5. IVR
  6. Greeting Message
  7. Business hours Call handling
  8. Call Recording
  9. Direct SMS
  10. Custom SMS sender Name
  11. Department SMS
  12. Direct FAX
  13. Department FAX
  14. Department Ring Group
  15. Deskphone Provisioning
  16. Softphone Provisioning
